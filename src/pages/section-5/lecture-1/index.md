# 5. Controlling Animation with JavaScript

## Learning Objectives for this Section
- Learn how to dynamically control animations: **sequential execution, pause, restart, and reverse**
- Understand how to **change animation speed** and implement animations that are consistent regardless of performance

- **Situation**: You want to control running animations interactively based on user actions.
- **Questions**:  
  - Can you pause or restart an animation with a button click?
  - Can you find out the progress of the animation?
  - Can you make the second animation start a few seconds after the first animation ends?

  - Is this possible with pure CSS, or do you need JavaScript?
- **Example**:  
  - An element moving to the right instantly reverses to the left when a button is clicked.
